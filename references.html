실행 순서

  1. Grok 리서치 프롬프트로 후보의 정의, 히스토리, 실제 사이트, 공식 출처를 먼저 확정한다.
  2. ima2 job manifest의 target path로 PNG 원본을 생성한다.
  3. npm run images:thumbs로 WebP preview를 만든다.
  4. 승인된 ISM만 assets/data/isms.json에 편입하고, 공식 시스템은 references로 유지한다.
  5. README, AGENTS, structure, devlog가 같은 구현 상태를 말하는지 확인한다.
